Secret Message Machine

Write a secret message and we’ll scramble it into cool code words! Your friend can paste the code words into the “Decrypt” page to read your secret again.

Make a Secret Message Read a Secret Message
Easy Fun For kids
1

Write

Type your message (anything from a sentence to a paragraph).

2

Encrypt

We use AES-GCM under the hood and then convert Base64 symbols to words from your wordlist.

3

Share

Send the wordified output. The receiver pastes it into Decrypt to get the original text.

What this is

  • A playful layer on top of AES that converts symbols into recognizable words.
  • Configurable mapping via a JSON wordlist.
  • Simple, single-page encrypt/decrypt UI.

What this isn’t

  • A replacement for real, audited cryptography practices.
  • A guarantee of secrecy if you reuse keys or share them improperly.
  • Production-grade key management—this is a demo by design.
